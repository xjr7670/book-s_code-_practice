### 8.2.3 以下代码的输出是什么？

```java
int[][] array = new int[5][6];
int[] x = {1, 2};
array[0] = x;
System.out.println("array[0][1] is " + array[0][1]);
```

```
array[0][1] is 2
```

int[][] z = {{1, 2}};
int[][] m = {{1, 2}, {2, 3}};
