### 7.2.1 如何声明一个数组引用变量，如何创建一个数组？

声明数组：

```java
数据类型[] 变量名;
```

创建数组：

```java
数组变量 = new 数据类型[长度];
```

### 7.2.2 什么时候为数组分配内存？

声明数组的时候不分配，在创建数组的时候才分配

### 7.2.3 下面代码的输出是什么？

```java
int x = 30;
int[] numbers = new int[x];
x = 60;
System.out.println("x is " + x);
System.out.println("The size of numbers is " + numbers.length);
```
