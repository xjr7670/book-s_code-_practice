### 1.9.1 使用行尾括号风格，将下面的程序根据程序设计风格和文档指南进行重新格式化。

```java
public class Test 
{
  // Main method
  public static void main(String[] args) {
  /** Display output */
  System.out.println("Welcome to Java");
  } 
}
```

```java
/** Display output */
public class Test {
    // Main method
    public static void main(String[] args) {
        System.out.println("Welcome to Java");
    }
}
```
