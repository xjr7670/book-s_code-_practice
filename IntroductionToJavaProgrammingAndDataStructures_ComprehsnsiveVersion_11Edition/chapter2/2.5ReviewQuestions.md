### 2.5.1 请指出并修改下面代码中的错误

```java
1 public class Test {
2   public static void main(String[] args) {
3     int i = k + 2;
4     System.out.println(i);    
5   }    
6 }
```

变量 k 在使用前应先声明并初始化。修正后如下：

```java
public class Test {
    public static void main(String[] args) {
        int k = 1;
        int i = k + 2;
        System.out.println(i);
    }
}
```
